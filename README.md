# Platform-Runner
3D Platform Runner with campaign and multiplayer gameplay.

# Core Mechanics
- Double Jump
- Crawl
- Spiderman Web
- Glide
- Wall run

# Powers
- Rasengan that goes forward
- Fire breath
- Ice path
- Tornado
- Earthquake
- Turtle shield
- Tsunami

# Obstacles
- Rocks falling down a cliff
- Wrecking Ball
- Wrecking Hammer
- Falling path

# Assets that can help
- Rigging Animation
