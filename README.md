# Platform-Runner
3D Platform Runner with campaign and multiplayer gameplay.

# Core Mechanics
- Double jump
- Crawl
- Hook on wall to swing
- Glide
- Wall run
- Wall jump

# Powers
- Rasengan that goes forward
- Fire breath
- Ice path
- Tornado
- Earthquake
- Turtle shield
- Tsunami
- Lightning bolt
- Black hole

# Obstacles
- Rocks falling down a cliff
- Wrecking Ball
- Wrecking Hammer
- Falling path

# Assets that can help
- Rigging Animation
